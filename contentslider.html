    How to make awesome content slider using html css and javascript.2020 latest way with source code?





    Step to do


    1.Make 2 files. [html & Css]


    2.Add Html structure.


    3.Link CSS file.


    4.Add container inside of body tag.


    5.Add 10+ div inside of container.


    6.Style all items.


    7.Add javascript bottom of the body

    Add HTML ...


    
           <!DOCTYPE html>
    <html lang="en">
    <head>
              <meta charset="UTF-8">
              <meta name="viewport" content="width=device-width, initial-scale=1.0">
              <title>Content slider using button</title>
              <link rel="stylesheet" href="divslider.css">
    </head>
    <body>
              <div id="container">
                        <div id="div1">

                        </div>

                        </div>
                        <div id="div1">

                        </div>
                        <button id="left" onclick="left()"></button>
                        <button id="right" onclick="right()"></button>
              </div>
              <div id="hideScrollbar">           
              </div>
              <script>
                        function left(){
                                  var container = document.getElementById("container");
                                  container.scrollBy(500,0);
                        }
                        function right(){
                                    var container = document.getElementById("container");
                                  container.scrollBy(-500,0);
                        }
              </script>
    </body>
    </html>


    Add CSS ...


    
              *{
              margin: 0;
              padding: 0;

    How to Make Navigation bar using HTML and CSS?





    File Structure


    1.HTML file Name:Responsiblenev.html

    2.CSS file Name :Responsive.css




    This is a awesome navigation bar for you website you can use this into your website free free free . You learn making this types of navigation bar just using basic html and css.Hope you enjoy this project . You have any question related code Ask Now.




    Add HTML ...


    
              <!DOCTYPE html>
    <html lang="en">
    <head>
    
                 <meta charset="UTF-8">
    
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    
                       <title>My responsive  navigation bar</title>
      
                        <link rel="stylesheet" href="style.css">
    
                         </head>
    <body>
